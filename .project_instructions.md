# VidEdge MVP - Projeto

## Visão Geral
SaaS para criadores de canais faceless no YouTube. Gera ideias de vídeos, roteiros otimizados e gerencia produção em Kanban.

## Stack Técnico
- App Router com layout dark premium
- PostgreSQL + Prisma ORM
- NextAuth.js v4 com credenciais (email/senha)
- LLM API (gpt-5.4-mini) para geração de ideias e roteiros com streaming
- TailwindCSS com tema escuro customizado

## Design Visual
- Background: hsl(0 0% 4%) (#0a0a0a)
- Cards: hsl(0 0% 7%)
- Primary/Accent: purple (#a855f7), blue (#3b82f6), emerald (#10b981)
- Dark mode forçado, sem alternância light/dark
- Fontes: DM Sans (body), Plus Jakarta Sans (display), JetBrains Mono (mono)

## Estrutura de Páginas
- /login, /register, /forgot-password - Auth público
- /dashboard - Métricas e projetos recentes
- /ideas - Lista de ideias; /ideas/new - Gerar ideias com IA (streaming)
- /scripts - Lista de roteiros; /scripts/[id] - Editor de roteiro
- /board - Kanban drag-and-drop com 7 colunas
- /projects - Lista com filtros; /projects/[id] - Detalhes do projeto
- /settings - Perfil

## Modelos Prisma
User, Account, Session, VerificationToken, VideoIdea, Script, Project

## API Routes
- /api/signup, /api/auth/login, /api/auth/[...nextauth]
- /api/dashboard (GET)
- /api/ideas (GET/POST/DELETE), /api/ideas/generate (POST streaming)
- /api/scripts (GET/POST), /api/scripts/[id] (GET/PUT/DELETE), /api/scripts/generate (POST streaming)
- /api/projects (GET/POST), /api/projects/[id] (GET/PUT/DELETE)
- /api/market-intelligence/search (POST) - YouTube search with Viral Score
- /api/market-intelligence/opportunities (GET/POST) - saved opportunities
- /api/market-intelligence/opportunities/[id] (GET/PUT/DELETE)
- /api/market-intelligence/analyze (POST) - AI opportunity analysis
- /api/market-intelligence/rpm (GET/PUT) - RPM benchmarks
- /api/market-intelligence/history (GET) - search history

## YouTube Market Intelligence Module
- YouTube Data API v3 integration (lib/youtube.ts)
- Viral Score calculation 0-100 (lib/viral-score.ts)
- Revenue Estimator with RPM benchmarks (lib/revenue-estimator.ts)
- Pages: /market-intelligence, /opportunities
- Prisma models: YouTubeSearch, YouTubeVideo, YouTubeChannel, NicheOpportunity, RPMBenchmark
- RPM benchmarks seeded for 13 niches
- Env var: YOUTUBE_API_KEY

## Content Pipeline Module (Esteira de Conteúdo)
- Route: /content-pipeline | Sidebar: "Esteira de Conteúdo" (Workflow icon)
- API: /api/pipeline (list/create), /api/pipeline/[id] (CRUD), /api/pipeline/generate (POST LLM)
- DB models: PipelineSession, ThumbnailConcept, PatternAnalysis, GrowthPlan
- Prompt library: lib/pipeline-prompts.ts (5 stage generators with YouTube data context)
- 5 stages: Viral Ideas → Pattern Analysis → Thumbnails → Script → Growth Plan
- Features: session persistence, stage navigation, YT data auto-collection, idea chaining, regeneration

## Regras
- Dados isolados por userId em todas as queries
- Streaming SSE para geração de ideias e roteiros
- Sidebar com navegação completa e logout
