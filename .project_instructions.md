# VidEdge MVP - Projeto

## Visão Geral
SaaS para criadores de canais faceless no YouTube. Gera ideias de vídeos, roteiros otimizados e gerencia produção em Kanban.

## Stack Técnico
- App Router com layout dark premium
- PostgreSQL + Prisma ORM
- NextAuth.js v4 com credenciais (email/senha)
- LLM API (gpt-5.4-mini) para geração de ideias e roteiros com streaming
- TailwindCSS com tema escuro customizado

## Design Visual
- Background: hsl(0 0% 4%) (#0a0a0a)
- Cards: hsl(0 0% 7%)
- Primary/Accent: purple (#a855f7), blue (#3b82f6), emerald (#10b981)
- Dark mode forçado, sem alternância light/dark
- Fontes: DM Sans (body), Plus Jakarta Sans (display), JetBrains Mono (mono)

## Estrutura de Páginas
- /login, /register, /forgot-password - Auth público
- /dashboard - Métricas e projetos recentes
- /ideas - Lista de ideias; /ideas/new - Gerar ideias com IA (streaming)
- /scripts - Lista de roteiros; /scripts/[id] - Editor de roteiro
- /board - Kanban drag-and-drop com 7 colunas
- /projects - Lista com filtros; /projects/[id] - Detalhes do projeto
- /settings - Perfil

## Modelos Prisma
User, Account, Session, VerificationToken, VideoIdea, Script, Project

## API Routes
- /api/signup, /api/auth/login, /api/auth/[...nextauth]
- /api/dashboard (GET)
- /api/ideas (GET/POST/DELETE), /api/ideas/generate (POST streaming)
- /api/scripts (GET/POST), /api/scripts/[id] (GET/PUT/DELETE), /api/scripts/generate (POST streaming)
- /api/projects (GET/POST), /api/projects/[id] (GET/PUT/DELETE)

## Regras
- Dados isolados por userId em todas as queries
- Streaming SSE para geração de ideias e roteiros
- Sidebar com navegação completa e logout
